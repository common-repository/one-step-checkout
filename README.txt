As per the research 1 in 10 people just go away after adding products to cart, only because the checkout process is very long and keeps the user navigating to different steps of checkout pages. Woocommerce one step plugin will help you to overcome all these hindrances and let your customer buy the product in a single step.

With this plugin installed in your store, users will be able update their cart within the checkout page. No need to go back to cart and update the number of products or delete any product from the list.

In a single checkout page, you can show billing details, order details, payment gateway options. This is how users will be able to see all important information related to order on a single page and place orders if they find everything good.

== Design Options: ==

Woocommerce One Step Checkout plugin offers you different design options like you want to show details in a single or double column. You can also add custom css as per the requirement and materialize your checkout page.

The design changes can be applied individually to both Checkout page layout and Billing details layout.

== How to apply one step checkout page setting in front?:- ==
Once all settings are done, you can simply paste the shortcode in your checkout page and your checkout process will work as a single page checkout. The shortcode is [osc_one_step_checkout].

== Benefits::- ==
* More selling due to straight checkout process
* Customer retention
* Skip cart and direct checkout page
* Update cart items within checkout page
* Reduce cart abandonment
* Remove items within checkout page

== Installation ==

Install and Activate

1. Unzip the downloaded ‘one-step-checkout’ zip file.
2. Upload the ‘one-step-checkout’ folder and its contents into the wp-content/plugins/ directory of your WordPress installation.
3. Activate One step checkout Page Checkout from Plugins page.
4. Use shortcode [osc_one_step_checkout] in your checkout page.
